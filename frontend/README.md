# Boekhouding SaaS - Frontend

Next.js TypeScript frontend voor de boekhoudapplicatie.

## Installatie

```powershell
npm install
```

## Configuratie

Kopieer `.env.local.example` naar `.env.local` en pas de API URL aan indien nodig:

```
NEXT_PUBLIC_API_URL=https://localhost:7001
```

## Development

Start de development server:

```powershell
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in je browser.

## Build

```powershell
npm run build
npm start
```

## Features

- ✅ TypeScript
- ✅ Next.js 14 App Router
- ✅ API integratie met ASP.NET Core backend
- ✅ Responsive design
- ✅ Health check monitoring
